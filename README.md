# 🧠 NovaMind AI — Modern RAG PDF Assistant

NovaMind AI is a modern AI-powered Retrieval-Augmented Generation (RAG) application that allows users to upload PDF documents and interact with them conversationally using advanced semantic search and LLM-based reasoning.

Built using:

* Streamlit
* LangChain
* ChromaDB
* Groq API
* Sentence Transformers

This project focuses on:

* semantic AI retrieval
* modern dark UI
* recruiter-grade interface
* conversational PDF intelligence
* production-style AI workflows

# ⚡ Installation

## 1. Clone Repository

```bash
git clone https://github.com/your-username/novamind-ai.git
```

```bash
cd novamind-ai
```

---

# 🔧 Create Virtual Environment

## Windows

```bash
python -m venv .venv
```

```bash
.venv\\Scripts\\activate
```

---

## Mac/Linux

```bash
python3 -m venv .venv
```

```bash
source .venv/bin/activate
```

---

# 📦 Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
```

Get your API key from:

https://console.groq.com/keys

---

# ▶️ Run The Application

```bash
streamlit run app.py
```

Application runs at:

```bash
http://localhost:8501
```
